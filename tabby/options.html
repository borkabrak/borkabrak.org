Options Tabby Accepts

  • tabs

    Required. This is a hash where the keys are the tab names, and the values are the urls that provide the corresponding content.

  • defaulttab

    This is the pane to open when the tab set is first created. If not present, no tab will be open by default.

  • doslide

    If true, a slide animation will be used to show panes. Default false.

  • hotkeys

    If true, allows hotkeys to be used for tab switching ('h' or 'p' for previous tab, 'l' or 'n' for next tab). Default false.
