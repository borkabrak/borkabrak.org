An Example

Make sure to include tabby.css and tabby.js in your html. Then the Tabby() function may be called like this: 

  Tabby('tabset_container',{
    tabs: {
      'welcome': 'welcome.html',
      'contact': 'contact.html',
      'links': 'otherlinks.html',
    },
    defaulttab: 'welcome'
  });
This will create three tabs, labelled 'welcome', 'contact', and 'links'. When clicked, the tab pane will show the corresponding content. When the tab set is first created, the 'welcome' tab will be shown by default.
